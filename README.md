### ⬡ [DiffCheck](https://github.com/beastt1992/DiffCheck) — 變更自動標記雲形線

**Auto-mark revision clouds for design changes**  
在同一張 DWG 裡框選新舊兩個區域，差異位置自動產生紅色雲形線。不用再手動圈。

`DFC` `DFCC` `DFCT`

---

### ◈ [DrawingHealthScore](https://github.com/beastt1992/DrawingHealthScore) — DWG 健康診斷

**Scan any DWG and get a health score out of 100**  
8 項逐一評分——未使用圖層、未清理 Block、檔案大小、Xref——一鍵修復，顯示節省了多少 MB。

`DHS` `DHSFIX`

---

### ◫ [SyncBlock](https://github.com/beastt1992/SyncBlock-AutoCAD) — 主 Block 同步

**One master block. Everything else syncs automatically.**  
修改主 Block，執行 SyncNow，消防、無障礙、面積計算等所有子 Block 立即更新。

`SyncNow` `SFM`

---

### ▣ [BatchPDF](https://github.com/beastt1992/autocad-batch-plot) — 批次匯出 PDF

**Batch export all title block frames to individual PDFs**  
專為台灣與亞洲常見的 Model Space 出圖流程設計。選取圖框、設定比例，一次匯出所有圖紙。

`BPDF`

## Installation · 安裝方式

```
1.  Download the .lsp file from any repo above
    從上方任一 repo 下載 .lsp 檔

2.  In AutoCAD, type APPLOAD and load the file
    在 AutoCAD 輸入 APPLOAD 並載入檔案

3.  Run the command listed above
    輸入上方對應指令執行
```

> 💡 Add to **Startup Suite** in APPLOAD for automatic loading every session.  
> 加入 APPLOAD 的 **Startup Suite**，每次開啟 AutoCAD 自動載入。
